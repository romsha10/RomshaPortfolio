# Romsha Ahlawat — Personal Data Science Portfolio

A premium, minimalistic, and futuristic portfolio website designed to showcase production-level data engines, machine learning pipelines, and algorithmic fairness research. Built with high performance, accessibility, and clean architecture in mind.

## Key Architectural Features

- **Futuristic Typographic System:** Designed using *Plus Jakarta Sans* for precise, legible technical copy and *Space Mono* for sharp, dashboard-style metrics and telemetry logs.
- **Hardware-Accelerated Metric Counters:** Custom JavaScript layout engines count up metrics dynamically as they scroll into view.
- **Interactive Tech Persona Engine:** A dynamic, loop-controlled typing system highlighting domain specializations in data science, fairness evaluation, and healthcare analytics.
- **Resilient Fluid Accessibility:** Animations and layout triggers scale down seamlessly and safely for users with `prefers-reduced-motion` enabled.
- **Fully Responsive Navigation:** Implements a custom, mobile-ready layout drawer with a performance-optimized responsive CSS breakpoint framework.

---

## Built With

- **HTML5** — Semantic markup structure optimized for web scrapers and screen readers.
- **CSS3** — Custom design layout featuring a bespoke deep-emerald variables system and fluid media query breakpoints.
- **JavaScript (ES6+)** — Asynchronous Intersection Observers for cascading scroll triggers and execution-pacing intervals.
